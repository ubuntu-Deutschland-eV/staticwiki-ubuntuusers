InterWiki-Links (IWL) sind eine einfache Methode, auf Informationen von anderen Wikis oder Portalen zu verweisen, ohne dabei die komplette URL angeben zu müssen. Ein weitere Vorteil ist, das externe Interwikilinks ein Symbol vor dem Link haben. So kann man schneller erkennen, wohin dieser Link verweist.

Um einen InterWiki-Link zu erstellen, benutzt man den folgenden Code:

[InterWikiName:Seitenname:Titel der Seite]

Seitenname entspricht der URL ohne das durch InterWikiName bereits gesetzte Präfix. "Titel der Seite" ist optional und kann weggelassen werden.

Eventuell vorkommende Doppelpunkte (:) in der URL muss man bei InterWiki-Links durch \: maskieren oder das Unicode-Zeichen %3A benutzen, um sie von der Wiki-Syntax zu unterscheiden. Beispiel:

[InterWikiName:Seitenname\:Teilseite:Titel der Seite]

Aus http://wiki.openstreetmap.org/wiki/DE:Beginners_Guide wird so [osm:DE\:Beginners_Guide:Beginners' guide] bzw. Beginners' guide. Ob man InterWiki-Links, wie bei externen Links im hiesigen Wiki üblich, um eine Landesflagge ergänzt, ist bisher nicht geregelt und bleibt dem jeweiligen Wiki-Autor überlassen.

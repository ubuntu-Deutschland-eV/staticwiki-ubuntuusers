Neuer Eintrag¶

Öffnet man die Seiten zum Editieren, findet man die vorhandenen Einträge innerhalb der Vorlagen. Ein Eintrag steht zwischen zwei Zeilen, die mit drei Plus-Zeichen gekennzeichnet sind: 

+++
Icon
Titel
Beschreibung
Links
+++

Wie man sieht hat ein Eintrag genau vier Zeilen, diese stehen für:

  • ein Icon (idealerweise mindestens 32px mal 32px im PNG-Format)

  • einen Titel

  • eine kurze Beschreibung (ohne Links)

  • Links zu Wiki-Artikel, Foren-Diskussion, Projekt-Seite, Ansprechpartner (in Wiki-Syntax)

Im Rohformat des Wiki sieht das beispielhaft so aus: 

{{{#!vorlage Projekte, "Ein Beispiel"
+++
Wiki/Icons/unknown.png
Beispiel-Titel
Dies ist eine kurze Beschreibung zum Beispiel. In ihr wird das Projekt und die Möglichkeiten mitzumachen vorgestellt. Auf Links in der Beschreibung wird verzichtet.
[:Startseite:Beispiel im Wiki], [topic:du-moechtest-einen-artikel-schreiben-oder-ver:Beispiel im Forum]
+++
}}}

Und so sieht das Ergebnis aus:

Ein Beispiel

Beispiel-Titel		 Dies ist eine kurze Beschreibung zum Beispiel. In ihr wird das Projekt und die Möglichkeiten mitzumachen vorgestellt. Auf Links in der Beschreibung wird verzichtet.

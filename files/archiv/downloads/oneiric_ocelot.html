Ich habe die Datei heruntergeladen, was nun?¶

Nach dem Download kann man gemäß dieser Anleitung (GNU/Linux, Mac OS X und Windows) die heruntergeladene Datei brennen. Unter Installation werden Schritt-für-Schritt Anleitungen zur Benutzung der CDs bereitgestellt und auch seltene Ausnahmefälle behandelt. Sollte es zu unerwarteten Schwierigkeiten kommen, hilft die Seite Ubuntu-CD Problembehebung bei der Lösung.

Checksumme überprüfen¶

Anhand eines Vergleichs der Prüfsumme der direkt heruntergeladenen Datei (dem ISO-Abbild) und der „Soll-Prüfsumme“ kann vor dem Brennen geprüft werden, ob die Datei wirklich fehlerfrei und in Ordnung ist. Wer bereits Ubuntu oder eine andere Linux-Version installiert hat, kann dazu md5sum verwenden. Windows-Nutzer können beispielsweise die Freeware-Programme DP Hash oder DPASHA nutzen. Diese Liste enthält die Prüfsummen der ISO-Abbilder, die mit der selbst ermittelten exakt übereinstimmen muss.

Außerdem kann sich die CD, sobald sie geladen ist, auch selbst überprüfen – was aber nichts nützt, wenn man bereits eine fehlerhafte CD gebrannt hat.

Hinweise¶

Editionen¶

Wenn hier auf dieser Seite von "Ubuntu" die Rede ist, dann ist nicht nur Ubuntu-Linux selbst gemeint, sondern auch dessen Editionen (offizielle Varianten):

  • Ubuntu - die ursprüngliche Ubuntu-Version mit der GNOME-Desktopumgebung. Ab Ubuntu 11.04 kommt Unity zum Einsatz.

  • Kubuntu - die zweite offizielle Version. Hier kommt KDE als Desktopumgebung zum Einsatz

  • Xubuntu - ein Community-Projekt, welches aber aktiv von Canonical (der Firma hinter Ubuntu) unterstützt wird. Hier kommt Xfce als Desktopumgebung zum Einsatz

  • Lubuntu - ebenfalls ein Community-Projekt, das erstmals mit Ubuntu 11.10 offiziell unterstützt wird. Als Desktopumgebung wird LXDE verwendet.

  • Server Edition - Wie der Name schon vermuten lässt, ist dies eine Version, welche speziell auf den Servereinsatz zugeschnitten ist. Die Paketauswahl ist für den Einsatzzweck optimiert und es wird keine Desktop-Umgebung / grafische Benutzeroberfläche installiert.

32-/64-Bit¶

  • 32-Bit: auch i386 oder x86. Für ältere Rechner und einige leistungsschwache Netbooks mit 32-Bit-Prozessoren

  • 64-Bit: auch amd64 oder x86_64. Für alle aktuelleren PCs, Notebooks und einige Netbooks (weitere Information zu Ubuntu als 64-bit-Version)

Direktlink / Torrent¶

Es gibt zwei Möglichkeiten, sich ein CD-Abbild herunter zu laden:

  • Direktlink - man lädt das Abbild als einzelne Datei "am Stück" von einem externen Spiegel-Server herunter

  • Torrent - Es kommt BitTorrent zum Einsatz, d.h. die Datei wird stückweise von verschiedenen Rechnern heruntergeladen. Man kann den Download jederzeit pausieren, ohne von vorne beginnen zu müssen, und es wird durch einen automatischen Prüfsummenabgleich sichergestellt, dass beim Herunterladen keine Fehler entstanden sind. Vor allem aber werden die Download-Server entlastet, da man von anderen Benutzern herunterlädt und weitere Benutzer später von einem selbst.

Desktop-CD / Alternate-CD¶

  • Desktop-CD - Live-CD, mit der das Linux-System vom CD-Laufwerk aus gestartet wird, erste Tests ohne Veränderungen am bestehenden System durchgeführt werden können und anschließend eine grafisch unterstützte Installation möglich ist.

  • Alternate-CD - CD mit rein text-basierter OEM-, Minimal- oder Experteninstallation. Diese Version enthält auch mehr Pakete aus dem main-Bereich und kann auch für Updates verwendet werden. Die "Alternate"-Version kann auch nützlich sein, wenn Ubuntu von der Desktop-CD nicht starten sollte.

Daneben gibt es noch eine Desktop-DVD, die bereits alle Sprachpakete enthält (normalerweise werden Sprachpakete während der Installation aus dem Internet heruntergeladen). Siehe Alternative downloads

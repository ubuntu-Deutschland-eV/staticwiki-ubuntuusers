Dieser Wiki-Artikel soll eine Hilfestellung zur Benutzung der Multimediatasten der Logitechtastatur „Logitech Media Keyboard Elite“ darstellen.

Sondertasten bekannt machen¶

Hinweis:

Die Tastatur muss über die PS/2-Schnittstelle an den Computer angeschlossen sein, ansonsten werden die Tasten nicht richtig erkannt.

Falls die Sondertasten nicht bestimmt werden können[3], muss der Tastencode der jeweiligen Sondertaste dem System noch bekannt gemacht werden. Dazu muss die Datei /etc/rc.local mit Rootrechten in einem Editor[2] geöffnet und um folgende Zeilen erweitert werden: 

setkeycodes e004 132 # Messenger key
setkeycodes e05a 218 # Mode F button

# Zoom-Tasten
setkeycodes 6b 235   # 100% key
setkeycodes 6a 234   # - Zoom key
setkeycodes e011 145 # + Zoom key

# Sondertasten ohne F-Modus
setkeycodes e03b 187 # F1 - Help
setkeycodes e03c 188 # F2 - Word
setkeycodes e03d 189 # F3 - Excel
setkeycodes e03e 190 # F4 - Finance
setkeycodes e03f 191 # F5 - Undo
setkeycodes e040 192 # F6 - Redo
setkeycodes e041 193 # F7 - Print
setkeycodes e042 194 # F8 - Save
setkeycodes e043 195 # F9 - "Programmable A"
setkeycodes e044 196 # F10 - "Programmable B"
setkeycodes e057 215 # F11 - "Programmable C"
setkeycodes e058 216 # F12 - "Programmable D"

Nachdem man die Änderungen übernommen hat, lädt man die Konfigurationsdatei über das Terminal[1] neu: 

sh /etc/rc.local

Einrichtung der Sondertasten¶

Mit Programmen wie beispielsweise xbindkeys oder Hotkeys kann man nun die Sondertasten mit Funktionen belegen. Als Tastencode benutzt man folgende Werte:

Sondertasten
Tastencode Sondertaste Tastencode Sondertaste Tastencode Sondertaste Tastencode Sondertaste
132 Messenger 187 F1 - Help 191 F5 - Undo 195 F9 - "Programmable A"
235 100 % 188 F2 - Word 192 F6 - Redo 196 F10 - "Programmable B"
234 - Zoom 183 F3 - Excel 193 F7 - Print 215 F11 - "Programmable C"
145 + Zoom 190 F4 - Finance 194 F8 - Save 216 F12 - "Programmable D"
218 F-Mode

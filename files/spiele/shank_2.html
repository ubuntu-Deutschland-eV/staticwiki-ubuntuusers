Shank 2 ist die Fortsetzung des kommerziellen Kampfspiels Shank, welches ebenfalls für Linux erschienen ist. Der zweite Teil baut auf dem bereits bekannten Kampfsystem auf und setzt ebenfalls auf den Comicstil. Shank verteidigt im zweiten Teil der Serie, mit allen ihm zur Verfügung stehenden Waffen, seine Familie und verschont keinen seiner Gegner.

Zur Steuerung wird ein Controller empfohlen. Das Spiel kann im Multiplayer-Modus an einem PC betrieben werden.

Installation¶

Steam¶

Nach dem Erwerb des Spiels über die Vertriebsplattform oder den Client kann dieses anschließend über letzteren installiert [4] werden.

The Humble Indie Bundle #7¶

Das Spiel aus der Aktion The Humble Indie Bundle #7 herunterladen und entpacken [1] - z.B. nach ~/Spiele/Shank2. Das Spiel kann aus dem Installationsverzeichnis heraus über shank2-bin gestartet [2] werden. Sofern gewünscht, kann ein Menüeintrag [3] erstellt werden.

Ubuntu Software-Center¶

Shank 2 kann über das Software-Center käuflich erworben werden. Dazu ist eine Registrierung bzw. ein Zugang über Ubuntu One erforderlich, welchen man sich aber auch während des Zahlungsprozesses über das Software-Center anlegen kann. Anschließend wird das Spiel automatisch heruntergeladen und installiert. Für zukünftige Updates wird eine neue Paketquelle hinzugefügt.

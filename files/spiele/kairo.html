Kairo - betrete die verlorene Welt. Erkunde die verlassenen Monumente, löse Rätsel, erwecke die seltsamen und alten Maschinen wieder zum Leben. Erkenne den wahren Zweck von Kairo und erfülle das Schicksal.

Kairo ist ein atmosphärisches 3D-Spiel, welches von Richard Perrin entwickelt wird. Untermalt wird das Spiel mit der Musik von Bartosz Szturgiewicz.

Installation¶

Entwicklerseite¶

Nachdem man das Spiel über lockeddoorpuzzle.com erworben hat, erhält man per Email den Link um das Spiel aus dem Humble Store herunterzuladen. Die Datei Kairo-Linux-VERSIONSNUMMER.tar.gz nach ~/Spiele/Kairo entpacken [1]. Das Spiel kann mit Kairo aus dem installationsverzeichnis heraus gestartet [2] werden. Auf Wunsch einen Menüeintrag [3] vornehmen.

Ubuntu Software-Center¶

Kairo kann über das Software-Center käuflich erworben werden. Dazu ist eine Registrierung bzw. ein Zugang über Ubuntu One erforderlich, welchen man sich aber auch während des Zahlungsprozesses über das Software-Center anlegen kann. Anschließend wird das Spiel automatisch heruntergeladen und installiert. Für zukünftige Updates wird eine neue Paketquelle hinzugefügt.

Einstellungen¶

Die Spieleinstellungen sind im Hauptmenü, welches man über die Taste Esc erreicht, über den Punkt "Options" zu erreichen.

Demo¶

Eine Demoversion des Spiels kann von lockeddoorpuzzle.com als Archiv heruntergeladen werden. Die Installation entspricht der Vollversion.

NightSky HD ist ein Geschicklichkeitsspiel. Der Spieler steuert eine Kugel mit den Tasten durch immer schwierigere Level.

Ein Tutorial führt Schritt für Schritt in das Spielgeschehen ein.

Tutorial Spielszene

Installation¶

Desura¶

Das Spiel kann über die Internetseite oder den Client zur Spieleliste hinzugefügt und gestartet werden [4].

Indie Royale / Humble Bundle¶

Die tar.gz-Datei aus der Aktion The Humble Indie Bundle #4 oder Indie Royale - the Difficult 2nd Bundle herunterladen und anschließend im Homeverzeichnis, z.B. nach ~/Spiele/NightSky, entpacken [1]. Über NightSkyHD bzw. NightSkyHD_64 kann das Spiel, aus dem Installationsverzeichnis heraus, gestartet [2] werden.

Auf Wunsch einen Menüeintrag [3] vornehmen.

Konfiguration¶

Die Konfiguration des Spiels beschränkt sich auf wenige Grundeinstellungen. Über "Settings" im Hauptmenü können die Soundeinstellungen und der Vollbild-/Fenstermodus angepaßt werden.

Probleme und Lösungen¶

64-Bit¶

Bei der Verwendung einer 64-Bit-Architektur muss ggf. das folgende Paket installiert [5] werden:

  • ia32-libs

mit apturl

Paketliste zum Kopieren: 

sudo apt-get install ia32-libs 

sudo aptitude install ia32-libs

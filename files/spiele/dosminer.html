Dosminer ist ein noch in Entwicklung befindliches Open-World-Spiel, in dem es darum geht, aus einer virtuellen Welt mit Hilfe einer anderen Person zu flüchten. Es wird seit dem 14.09.2011 von der FRED games AG programmiert. Am 03.11.2013 wurde ein Neubeginn beschlossen und die erste Version in Java veröffentlicht. Seit diesem Tag steht das Spiel nicht nur für Windows, sondern auch für Linux zur Verfügung. Seit dem 21.12.2014 wird an der Neuauflage gearbeitet, die die Einbindung von Grafiken beinhaltet.

Die Idee des Spiels ist es, aus einer virtuellen Welt zu entkommen, indem man Items verwendet, um Systemfehler auszulösen. Diese zwar beschränken die Welt, die Vielfalt des Spiels wird jedoch vergrößert. Auf einer actionreichen Jagd geht es durch verschiedene Terrains und mithilfe von neuen Waffen lassen sich Gegner töten, die einem im Wege stehen könnten. Leider ist diese Spielidee in der grafischen Version noch nicht im Ansatz zu erkennen. Die Geschichte des Spiels erfährt man nur durch Äußerungen des Entwicklers LeMas und der Spielgeschichte der älteren, terminalbasierenden Versionen.

Voraussetzungen¶

Zur Nutzung von Dosminer ist Java erforderlich. Die Installation ist im Artikel Java/Installation beschrieben.

Installation¶

Von itch.io ist die Archivdatei alpha snapshot s_VERSION.zip herunterladen und entpacken [1]. Danach kann man die beim Entpacken entstandenen Dateien in ein beliebiges Verzeichnis verschieben, zum Beispiel nach ~/Spiele/Dosminer im Homeverzeichnis. Aus diesem Ordner kann das Spiel mit folgendem Befehl gestartet werden [2] [3]:

java -jar Dosminer.jar

Bei Bedarf kann auch ein Programmstarter [4] angelegt werden.

Es ist auch möglich, die Dosminer.jar direkt mit Java zu öffnen, hierbei kann es nötig sein, dass sämtliche .wav-Dateien ins Homeverzeichnis verschoben werden.

Steuerung¶

Die gesamte Steuerung kann man unter dem Menüpunkt "Help" ansehen. Es wird folgende Steuerung für den ersten Spieler verwendet:

Tasten zur Spielsteuerung
Taste Funktion
Bewegen
Q E Schießen
R F Inventar öffnen/schließen
X Item aufnehmen
E Item benutzen (im Inventar)
         V Pausemodus an/aus
T G 2-Spieler-Modus an/aus

Die Tastenbelegung kann nicht geändert werden, und die Belegung von 2 Tasten auf eine Funktion (Pause an/aus; Inventar öffnen/schließen) kann auf Dauer als störend empfunden werden, ist nach einiger Einspielzeit nicht mehr sehr tragisch.

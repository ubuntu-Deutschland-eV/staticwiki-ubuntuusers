Majesty Gold Edition ist ein Echtzeitstrategiespiel. Es enthält neben dem Basisspiel "Majesty: The Fantasy Kingdom Sim" die Erweiterung "Majesty: The Northern Expansion". Die Linux Version wurde unter dem ursprünglichen Titel des Hauptprogramms "Majesty: The Fantasy Kingdom Sim" veröffentlicht.

Für die Installation liegt ein Installer von LGP auf der Spiele-CD bereit. Da dieser auf neueren Systemen nicht einwandfrei funktioniert benötigt man die aktuelle Installationsroutine new_installer.run von holarse-linuxgaming.de .

Hauptmenü Spielszene

Installation¶

CD-Version¶

Um Majesty zu installieren [2] [3], führt man das Skript new_installer.run im Terminal [1] aus: 

./new_installer.run

Bei der Installation den Anweisungen folgen. Das Spiel wird im angegebenen Ordner installiert.

Update¶

Das Vorgehen hierzu ist im Artikel LGP beschrieben.

Manuelles Update¶

Updates findet man unter holarse-linuxgaming.de und patches-scrolls.com . Diese von der jeweiligen Homepage herunterladen und wie gewohnt installieren. Nacheinander die folgenden Updates einspielen [3]:

  • majesty-1.4-1.4.01.run

  • majesty-1.4.01-1.4.02-x86.run

  • majesty-1.4.02-1.4.03-x86.run

Desura¶

Das Spiel kann über die Internetseite oder den Client zur Spieleliste hinzugefügt und gestartet werden [4].

Ubuntu Software-Center¶

Majesty Gold kann über das Software-Center käuflich erworben werden. Dazu ist eine Registrierung bzw. ein Zugang über Ubuntu One erforderlich, welchen man sich aber auch während des Zahlungsprozesses über das Software-Center anlegen kann. Anschließend wird das Spiel automatisch heruntergeladen und installiert. Für zukünftige Updates wird eine neue Paketquelle hinzugefügt.

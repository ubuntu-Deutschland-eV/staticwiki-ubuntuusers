Spirits ist ein kommerzielles Puzzle-Spiel. Im Herbst fallen die Blätter von den Bäumen. Aus diesen steigen die Geister empor um ihre Heimreise zu beginnen. Durch den Einsatz der zur Verfügung stehenden Aktionen, welche mit Bedacht gewählt wurden, hilft der Spieler den Geistern ihren Weg zu finden. Manch einer wird sich für die anderen Geister opfern…

Die Spielsteuerung erfolgt über die Maus. Eine Highscore-Tabelle kann online eingesehen werden.

Menü Spielszene

Installation¶

The Humble Android Bundle #3¶

Das zur Rechnerarchitektur passende .deb-Paket aus der Aktion The Humble Android Bundle #3 herunterladen und installieren [1].

Das Spiel kann über "Spiele -> Spirits" gestartet werden bzw. unter Unity über den Aufruf von "Spirits".

Ubuntu Software-Center¶

Spirits kann über das Software-Center käuflich erworben werden. Dazu ist eine Registrierung bzw. ein Zugang über Ubuntu One erforderlich, welchen man sich aber auch während des Zahlungsprozesses über das Software-Center anlegen kann. Anschließend wird das Spiel automatisch heruntergeladen und installiert. Für zukünftige Updates wird eine neue Paketquelle hinzugefügt.

Einstellungen¶

Sofern man die Übermittlung der Highscores unterbinden möchte deaktiviert man beim Spielstart im Launcher die Option "Online Highscores". Alternativ im Hauptmenü den Punkt "Options" aufrufen und hier bei den Online Highscores "No" auswählen.

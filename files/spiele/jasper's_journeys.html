Jasper's Journeys ist ein kommerzielles Jump&Run-Spiel. Orlando, Jasper's Katze, wurde von der bösen Hexe, entführt. Diese beabsichtigt, das Tier als Zutat für einen Zaubertrank zu verwenden. Jasper stürzt sich ins Abenteuer, um Orlando aus dieser misslichen Lage zu befreien. Auf dem Weg zum Schloss gibt es Unmengen an Schätzen, Kristallen und anderen Items in den Leveln zu sammeln... Spielstände können in den Inns gesichert werden. Dort kann man auch weitere nützliche Dinge für das Spiel kaufen...

Hauptmenü Intro Spielszene

Installation¶

Das Spiel aus der Aktion The Humble Voxatron Debut oder von der Entwicklerseite als .deb-Paket herunterladen und installieren [1].

Über "Anwendungen -> Spiele -> Jasper's Journeys" kann das Spiel gestartet werden.

Bedienung¶

Das Spiel kann wahlweise mit der Tastatur oder einem Controller gespielt werden.

Menü
Menüpunkt Beschreibung
"Play" Spiel starten
"Load Game" Spielstand laden. Spielstände können im Spiel "erworben" werden.
"Instructions" Spieleinführung.
"Options" Tastenbelegung ändern, Sound (de)aktivieren, Vollbild-/Fenstermodus und Joystick aktivieren.
"Exit" Spiel beenden.

Auf dem Planeten Terra Nova herrscht Krieg. Als die Stadt Peace River vollständig zerstört wird, ohne dass eine der verfeindeten Supermächte daran schuld ist, wird klar, dass eine feindliche Invasion des Planeten bevorsteht. Eine Koalition der Supermächte kann das Schlimmste verhindern... In Heavy Gear II spielt man den Kommandanten einer aus Robotern (Gears) bestehenden Elite-Einheit, die mit Kampfanzügen ausgestattet sind. Modernste Technologie und Waffen sind erforderlich um ...

Installation¶

Zur Installation wird eine aktuelle Routine von liflg.org oder holarse-linuxgaming.de benötigt. Diese nach dem herunterladen ausführbar [2] machen und die den Vorgang einleiten [3]: 

./hg.2_1.00b-english_x86.mojo.run

Das Spiel kann man anschließend über das Menü ("Spiele -> Heavy Gear II") starten. Beim ersten Start wird der Ordner ~/.loki/hg2 im Homeverzeichnis angelegt um Spielstände et cetera dort abzulegen.

Alternativ kann es über hg2 im Installationsverzeichnis aufgerufen [4] werden. Weitere Startparameter, die auch kombiniert genutzt werden können:

Startoptionen
ParameterBeschreibung
-wSpiel startet im Fenstermodus.
-sSoundwiedergabe wird gestoppt.
-gOpenGL-Effekte aktivieren.
-hweitere Optionen

Netzwerk¶

Von icculus.org die Datei servers.dat herunterladen und die Datei aus ~/.loki/hg2/ damit ersetzen.

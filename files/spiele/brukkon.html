Brukkon (Althochdeutsch: Eine Brücke schlagen) ist ein kommerzielles Puzzle-Spiel aus dem Hause 93-interactive, welches seit Oktober 2015 kostenlos zur Verfügung steht. Man steuert einen kleinen Roboter, welcher Ersatzteile benötigt, in möglichst wenigen Zügen über die Plattformen zum Ziel. Das Spiel beinhaltet 30 herausfordernde Level, in denen der Schwierigkeitsgrad immer weiter zunimmt. Über die Highscore-Tabelle kann man sich mit anderen Spielern messen.

Der Ubuntu-Store bietet eine spezielle Variante mit 5 Extra-Levels an.

Installation¶

Entwicklerseite¶

Das Spiel kann von der Entwicklerseite als .deb-Paket heruntergeladen werden. Dieses wie gewohnt installieren [1]. Anschließend kann das Spiel über "Anwendungen -> Spiele" gestartet werden.

Alternativ steht ein .tar.gz-Archiv zur Verfügung, welches in den Spieleordner entpackt [2] wird, z.B. ~/Spiele/brukkon.

Über Brukkon kann das Spiel aus diesem heraus gestartet [3] werden. Auf Wunsch einen Menüeintrag [4] vornehmen.

Extra-Level

Ubuntu Software-Center¶

Brukkon kann über das Software-Center käuflich erworben werden. Diese Variante beinhaltet - im Gegensatz zur Ursprungsversion - insgesamt 35 Level. Dazu ist eine Registrierung bzw. ein Zugang über Ubuntu One erforderlich, den man sich aber auch während des Zahlungsprozesses über das Software-Center anlegen kann. Anschließend wird das Spiel automatisch heruntergeladen und installiert. Für zukünftige Updates wird eine neue Paketquelle hinzugefügt.

Bedienung¶

Nach dem Spielstart stehen im Hauptmenü folgende Punkte zur Verfügung:

Hauptmenü
Menüpunkt Beschreibung
"Play" Spielstart.
"Instructions" Videotutorial, welches in die Feinheiten des Spiels einführt.
"Options" Einstellungen an der Darstellung und der Lautstärke vornehmen.
"Exit" Spiel beenden.

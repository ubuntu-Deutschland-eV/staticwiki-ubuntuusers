Zen Puzzle Garden

Zen Puzzle Garden ist ein kommerzielles Denkspiel. Man steuert einen Zen-Mönch. Sobald dieser die Sandfläche des Kare-san-sui betritt, recht er geradeaus weiter, bis er diese Fläche verlässt oder gegen einen Stein/Felsbrocken stößt. Hier bestimmt der Spieler die Richtung, in die der Mönch weitergehen soll. Das Puzzle ist gelöst, wenn die ganze Sandfläche gerecht wurde. Jedoch kann der Mönch bereits gerechte Flächen nicht erneut betreten.

Ein Editor erlaubt es, neue Level zu entwerfen.

Spielszene

Installation¶

Das Spiel aus der Aktion The Humble Voxatron Debut oder von der Entwicklerseite als .deb-Paket herunterladen und installieren [1].

Über "Anwendungen -> Spiele -> Zen Puzzle Garden" kann das Spiel gestartet werden.

Bedienung¶

Das Spiel kann wahlweise mit der Tastatur oder der Maus gespielt werden.

Menü
Menüpunkt Beschreibung
"Play" Puzzle starten. Bereits gelöste Levels werden hervorgehoben.
"Garden Designer" Startet den Leveleditor.
"Options" Audioeinstellungen, Fenster-/Vollbildmodus und Farbgebung des Charakters.
"Exit" Spiel beenden.
"Zen Online" Dieser Link funktioniert nicht mehr. Hier BBS im Browser aufrufen.

Editor¶

Editor

Mit dem Leveleditor können eigene Kreationen für das Spiel geschaffen werden. Nach einer Registrierung kann man diese im Forum hinterlegen . Diese stehen anderen Spielern im Forum zur Verfügung und können dort gespielt werden. Eine Liste von Kreationen kann unter lexaloffle.com eingesehen werden.

Erstellte Level sind im Homeverzeichnis unter ~/.lexaloffle/Zen Puzzle Garden/gardens zu finden. Der Code für den im Beispielbild hinterlegten unspielbaren Garten lautet:

Tastenkürzel¶

Steuerung
Taste(n)Beschreibung
↑ + ↓ + ← + → Mönch bewegen.
R Puzzle neu beginnen.
U Den letzten Spielzug zurücknehmen.
Esc Menü aufrufen.
F Schnellen Modus (de)aktivieren,
I Sofort-Modus ein- bzw. ausschalten.
S Sound ein/aus.
P Bildschirmfoto erstellen. Dieser wird auf dem Desktop als tga gespeichert.

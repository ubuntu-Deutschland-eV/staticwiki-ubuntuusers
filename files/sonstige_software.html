Sonstige Software

Auf dieser Seite werden Artikel gesammelt, die zur Zeit keiner bestehenden Übersichtsseite zugeordnet werden können.

Dateimanagement¶

  • Dateimanager Dateien und Ordner im Dateisystem verschieben, kopieren, löschen usw.

  • Desktopsuchmaschinen Dateien auf dem Computer suchen

  • Virenscanner Dateien auf Viren überprüfen

  • Packprogramme Programme und Formate für komprimierte Dateien

  • CD-Images ISO-Images ins Dateisystem einbinden

    • AcetoneISO - wie CDEmu, beherrscht aber weniger Formate

    • CDEmu - verschiedene Image-Dateien und -Formate als virtuelles Laufwerk einbinden

  • cuteRenamer - einfach zu bedienendes Programm zum schnellen Umbenennen von Dateien

  • GPRename - Umbenennen von Dateien und Verzeichnissen

  • H2rename - Umbenennen von Dateien, mit dem Schwerpunkt Zeichensatzprobleme

  • rename - Dateien eines Ordners mithilfe der regulären Ausdrücke von Perl umbenennen

  • Krename - Dateien einfach umbenennen

  • pyRenamer - leistungsstarkes und mächtiges Programm zum Umbenennen von Dateien

  • LXSplit - Dateisplitter

  • Cover Thumbnailer - Inhaltsanzeige (statt einfacher Ordnersymbole) für den Dateimanager Nautilus

  • DupeGuru - doppelte Dateien finden

  • fslint - Dateisystem-Pflege

  • Zeichensatz-Konverter - für Dateinamen und Inhalte

Editoren¶

  • Editor - Standardprogramme zum Bearbeiten von Textdateien

  • Editoren Programmübersicht

    • Webeditoren für HTML, PHP, etc.

    • LaTeX-Editoren Hilfsmittel für LaTeX

    • Entwicklungsumgebungen Programmierung

OpenStreetMap und Navigation¶

  • CacheWolf - Hilfsprogramm zum Geocaching

  • OpenStreetMap - freie Weltkarte

    • Programme - Editoren, Konverter ...

    • JOSM - Java-OpenStreetMap-Editor

    • OSM2World - OSM-Karten im 3D-Modus darstellen

  • Navit - Navigationssoftware, die auch mit OSM-Karten umgehen kann

  • GpsDrive - GPS-Navigation

  • FoxtrotGPS - einfach zu nutzendes Programm zum Arbeiten mit Karten

Emulation und Virtualisierung¶

  • Virtualisierung Überblick über die verschiedenen Techniken und Programme

    • KVM - Kernel Virtual Machine, die Virtualisierungslösung, welche direkt im Kernel implementiert ist

    • QEMU - universeller und flexibler Virtualisierer und Emulator

    • VirtualBox - proprietäre Virtualisierungslösung

    • VMware Player- Artikel zum Player von VMware

    • virt-manager - grafischer Oberfläche zur Verwaltung verschiedener virtueller Maschinen

    • virsh - Werkzeug zum Verwalten virtueller Maschinen

  • Wine - Windows-Programme unter Linux installieren und nutzen

    • CrossOver - auf bestimmte Windows-Büroanwendungen optimierte Version von Wine

    • Playonlinux - Nutzung unterschiedlicher Wine-Version auf einem Rechner

    • Windows-Programme - Tipps zur Nutzung von Wine

  • andLinux - Linux-Programme unter Windows nutzen

  • Dosemu - DOS-Emulator

  • DOSBox - noch ein DOS-Emulator

Verteiltes Rechnen¶

  • BOINC - Software-Plattform für verteiltes Rechnen, u.a. SETI@home

Sonstige ohne Kategorie¶

  • 64-Bit-Architektur - Grundlagenartikel zur den Fähigkeiten moderner Prozessoren

  • Amateurfunk Übersichtsartikel

    • CQRLOG - ein Logbuch-Programm für Funkamateure

    • Gqrx - Funkempfänger-Software

  • Andere Distributionen - ein Blick über den Tellerrand auf weitere Linux-Distributionen

  • Bitcoin-QT - ein Wallet für die Währung Bitcoin

  • Cura - 3D-Daten für den 3D-Druck vorbereiten

  • cycle - Menstruationszyklus protokollieren und voraussagen

  • Electric Sheep - Projekt zum verteilen Rechnen zur Berechnung komplexer Grafiken

  • Fahrerkarte unter Linux - Auswertung von Fahrerkarten

  • Gourmet Rezept Manager - Rezepte komfortabel erfassen und verwalten

  • Aufsetzen eines Gastro-Kassensystems mit OrderSprinter

  • Kiwix - die Wikipedia und das ubuntuusers-Wiki offline nutzen

  • OpenAstro.org - Horoskope erstellen

  • QuitCount - Hilfsmittel, um mit dem Rauchen aufzuhören

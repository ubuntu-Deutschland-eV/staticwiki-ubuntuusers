Achtung!

Es kann bei einigen Laptop Modellen, unter anderem von Lenovo, Toshiba und Acer, zu Problemen mit dem BIOS des Rechners kommen, welche im schlimmsten Fall dazu führen können, dass der Rechner nicht mehr bootet. Dies kann sowohl bei der Installation von Ubuntu 17.10. als auch schon bei der Nutzung der Live-CD passieren!

Die nächste Ubuntuversion nach Zesty Zapus heißt 17.10 „Artful Aardvark“. Diese Version wird neun Monate mit Aktualisierungen versorgt. Der Zeitplan wird auf den Developer Summits festgelegt.

Kernel¶

Ubuntu 17.10 wird mit Kernel 4.13 ausgeliefert.

System¶

Bei Ubuntu 17.10 kommt Wayland als Display Server zum Einsatz, alternativ kann Ubuntu aber auch weiterhin über den X-Server laufen.

Zur Konfiguration des Netzwerks bei Ubuntu Server wird nicht mehr die Datei /etc/network/interfaces verwendet, sondern das von Canonical entwickelte netplan, welches über YAML-Dateien, die in /etc/netplan konfiguriert wird

cifs: SMB3 ist jetzt der Standard, vorher SMB1. Für ältere Shares muss nun beim Mounten ggf. die passende SMB Version z.B. vers=1.0 als Option verwendet werden.

Programmaktualisierungen¶

Firefox liegt in der Version 56 vor, Thunderbird in Version 52.4. LibreOffice ist in Version 5.4 enthalten.

Offizielle Varianten¶

Ubuntu¶

Ubuntu 17.10 nutzt bei Artful Aardvark wieder GNOME (in der Version 3.26.1) als Desktop, Unity kann bei Bedarf über die Paketquellen nachinstalliert werden. Mit dem Wechsel zurück zu GNOME wird auch wieder GDM als Display Manager eingesetzt, was den bisher eingesetzten Display Manager LightDM ersetzt.

Des Weiteren gibt es für Artful keine 32-bit Desktop-Images mehr. Die 32-bit Pakete werden jedoch weiter gepflegt. Die Server-Version gibt es auch weiterhin als 32-bit Version, ebenso wie einige der offiziellen Derivate.

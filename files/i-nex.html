I-Nex

Dieser Artikel wurde für die folgenden Ubuntu-Versionen getestet:

  • Ubuntu 16.04 Xenial Xerus

  • Ubuntu 14.04 Trusty Tahr

Zum Verständnis dieses Artikels sind folgende Seiten hilfreich:

  1. Verwenden eines PPAs

  2. Installation von Programmen

  3. Programme starten

Inhaltsverzeichnis
  1. Installation
  2. Bedienung
  3. Links

I-Nex ist ein Werkzeug, mit dem wie bei CPU-Z die Daten des Prozessors und andere Hardware-Informationen ausgelesen und über eine grafische Oberfläche angezeigt werden können. Entwickelt wurde die Anwendung mit Gambas. Die Sprache der Benutzeroberfläche ist Englisch, um einen möglichst universellen Einsatz zu erlauben.

Alternativen zu I-Nex sind die Programme Sysinfo, HardInfo und CPU-G. Die ersten beiden genannten berücksichtigen ebenfalls weitere Systemkomponenten wie Grafikkarten, Massenspeicher, Schnittstellen etc.

Installation¶

Das Programm ist nicht in den offiziellen Paketquellen enthalten. Daher muss man auf ein "Personal Package Archiv" (PPA) [1] ausweichen.

PPA¶

Adresszeile zum Hinzufügen des PPAs:

  • ppa:i-nex-development-team/stable

Hinweis!

Zusätzliche Fremdquellen können das System gefährden.

Ein PPA unterstützt nicht zwangsläufig alle Ubuntu-Versionen. Weitere Informationen sind der PPA-Beschreibung des Eigentümers/Teams i-nex-development-team zu entnehmen.

Damit Pakete aus dem PPA genutzt werden können, müssen die Paketquellen neu eingelesen werden.

Nach dem Aktualisieren der Paketquellen kann folgendes Paket installiert [2] werden:

  • i-nex (ppa)

mit apturl

Paketliste zum Kopieren: 

sudo apt-get install i-nex 

sudo aptitude install i-nex

Hinweis:

Bei der Installation über das PPA wird unter Ubuntu 12.04 eine Reihe von Gambas3-Paketen mitinstalliert, da die Version 3 der Programmiersprache hier noch nicht in den offiziellen Paketquellen enthalten ist (erst ab Ubuntu 12.10).

Bedienung¶

Nach der Installation steht das Programm bei Ubuntu-Varianten mit einem Anwendungsmenü unter "System -> I-Nex" zur Verfügung [3]. Zur Benutzung selbst gibt es nicht viel zu erklären. Die Informationen sind auf verschiedene Registerkarten verteilt:

  • "CPU" - Prozessor (Central Processing Unit)

  • "GPU" - Grafikkarte (Graphics Processing Unit)

  • "Mobo" - Motherboard (Chipset- und BIOS-Informationen)

  • "Audio" - Soundkarte(n)

  • "Drives" - Massenspeicher

  • "System" - Rechnername, Systemarchitektur, Ubuntu-Version

  • "Kernel" - Kernel-Version(en)

  • "Memory" - Arbeitsspeicher

  • "Network" - Netzwerk-Informationen

  • "Battery" - Akku-Informationen (nur bei Notebooks)

  • "USB | Input" - Angaben zum Universal Serial Bus

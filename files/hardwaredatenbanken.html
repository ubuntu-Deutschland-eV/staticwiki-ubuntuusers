Von Linux nicht unterstützte Hardware wird zum Glück immer seltener. Trotzdem gibt es immer wieder Inkompatibilitäten, insbesondere bei ganz neuer oder exotischer Hardware. Für denjenigen, der sich eine neues Gerät anschaffen möchte, gilt es daher sich vorher schlau zu machen. Meistens wurde das gewünschte Gerät schon von jemandem gekauft, unter Linux ausprobiert und die Erfahrungen in einer Hardwaredatenbank festgehalten. Eine andere Variante ist die Frage, ob ein älteres Gerät unter Linux unterstützt wird.

Hier sind nun externe Hardwaredatenbanken aufgeführt, die einem bei der Frage, ob ein Gerät funktioniert oder nicht, unterstützen können.

Hardwaredatenbanken¶

Speziell für Ubuntu¶

  • Component catalog - Von Canonical zertifizierte Herstellerliste bestimmter Hardware

  • Ubuntu-certified hardware - Zertifizierte Modelle von Computerherstellern, teils mit Hinweisen zur Installation

  • Ubuntu – Unterstützte Laptops

  • Free Software Drivers - (veraltet) Verschiedene Hardware-Hersteller bewertet auf einer Skala von 1-10, je nach dem wie gut die Zusammenarbeit bei freien Linux-Treibern in der Praxis klappt.

  • Debian GNU/Linux Gerätetreiber Überprüfungsseite - Komfortable Treiberprüfung für Debian-Systeme

  • Linux Mint Hardware Database

Hardware unter Linux allgemein¶

  • h-node - die Free Software Foundation unterhält eine weitere Datenbank mit Hardware, die GNU/Linux unterstützt

  • LiNUWARE - auf dieser Plattform hat man die Möglichkeit, die GNU/Linux-Kompatibilität der eigenen Hardware zu bewerten und zu kommentieren

  • Linux-drivers.org - Linkliste, die viele Distributionen abdeckt

  • Tuxhardware - Hardware, die garantiert unter Linux läuft

  • Linux-Hardware-Guide.de - Datenbank aktueller Hardware untersucht auf Linux-Kompatibilität mit Marktpreisen und Preisentwicklung. Mit einfachem Script um selber die eigene Hardware zuzufügen.

  • Installing Debian On - im Aufbau

Spezielle Hardware¶

Laptops¶

  • TuxMobil.de - Linux mit Laptop, PDA & Handy

  • TuxMobil.org - das umfangreichere englische Original

  • Linux on Laptops - Reports on running Linux on notebook or laptop computers

  • ThinkWiki - speziell für IBM Thinkpads

  • ThinkWiki.de - nicht ganz so umfangreich wie das englische Vorbild, aber auf Deutsch

Drucker¶

  • Open Printing - Drucker-Datenbank (Linux Foundation)

TV-Karten¶

  • linuxtv.org - V4L, DVB und Komponenten

Scanner¶

  • SANE Supported Devices - liefert auch Informationen (soweit vorhanden), wenn nicht unterstützt

Soundkarten¶

  • ALSA Hardware Device Information - ALSA SoundCard Matrix

Modems¶

  • Linmodems - Einsatz von Modems unter Linux

Links¶

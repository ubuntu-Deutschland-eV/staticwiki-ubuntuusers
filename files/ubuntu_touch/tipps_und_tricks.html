Tipps und Tricks

Dieser Artikel bezieht sich auf Ubuntu Touch, die Variante von Ubuntu für mobile Geräte mit Touchscreen.

In diesem Artikel werden einige Tipps und Tricks gesammelt, welche für den Umgang mit Ubuntu Touch hilfreich sein können.

Es sollte beachtet werden, dass das Betriebssystem einem ständigen Wandel unterliegt und der ein oder andere Hinweis in neueren Releases obsolet ist.

Bildschirmfoto¶

Durch gleichzeitiges Drücken der beiden Lautstärketasten Vol+ + Vol- wird ein Bildschirmfoto aufgenommen und abgespeichert. Dieses ist im Ordner Pictures/Screenshots der Ordnerstruktur des Homeverzeichnisses bzw. der Micro-SD-Karte zu finden.

Dateimanager¶

Die Dateiverwaltung aus den Ubuntu Core Apps[1] muss erst im Ubuntu Store nachinstalliert werden. Rechts oben auf das Menü tippen und "Für vollen Zugriff" anwählen, um Root-Rechte erlangen zu können, falls diese benötigt werden. Momentan ist dies auch für den Zugriff auf Dateien auf der Micro-SD-Karte oder per USB-Adapter angesteckten USB-Sticks nötig. Diese externen Medien erreicht man danach, indem man den unteren Bildschirmrand ("Places" bzw. "Orte") nach oben zieht.

Klingeltöne¶

Zur Verwendung eigener Benachrichtigungs- / Klingeltöne muss zuerst das Ubuntu Touch Terminal installiert werden.

Das System muss temporär beschreibbar[2] gemacht werden. Anschließend werden die Töne mit Root-Rechten in das richtige Verzeichnis kopiert:

sudo cp /path/to/your/ringtone/ringtone.ogg /usr/share/sounds/ubuntu/ringtones/ # Klingelton
sudo cp /path/to/your/ringtone/ringtone.ogg /usr/share/sounds/ubuntu/notifications/ # Benachrichtigungen

Die Töne können nun in "Systemeinstellungen → Audio" ausgewählt werden.

Hinweis:

Dieser Schritt ist seit der Veröffentlichung von OTA-9 nicht mehr nötig.

Speicherort von SMS¶

SMS werden in der Datei ~/.local/share/history-service/history.sqlite gespeichert. Sie können angezeigt werden, indem man die Datei aufruft, wie in den Beispielen von SQLite erläutert wird.

Nutzungsdauer von Apps¶

Die Nutzungsdauer von Apps lässt sich mit folgendem Befehl[2] abrufen: 

ubuntu-app-usage

Terminal¶

Um in der Terminal App die Autovervollständigung zu nutzen ( Tab ⇆ auf dem PC), tippt man während der Eingabe mit dem Finger auf den Bildschirm. Wischt man mit zwei Fingern nach unten bzw. oben, kann man durch bereits ausgeführte Befehle scrollen. Dasselbe mit nur einem Finger scrollt die Terminal-Seite nach oben bzw. unten, allerdings scrollt man stattdessen ebenfalls nur durch alte Befehle, wenn zu wenige Terminal-Ausgaben zum Scrollen vorhanden sind.

Hinweis:

Unter Terminal sind mehrere Möglichkeiten aufgeführt, welche den Zugriff auf das Ubuntu Touch-Gerät gestatten.

Ubuntu Touch Apps am Ubuntu PC installieren¶

Man kann Ubuntu Touch Apps auch auf Ubuntu testen. Diese sind in Form von .click-Dateien vereinzelt in Fremdquellen verfügbar, z.B. als dekko.click für die Email App Dekko. Man installiert sie wie in folgendem Beispiel: 

click install --allow-unauthenticated --force-missing-framework --user=$USER dekko.click

Die Optionen sind nicht immer nötig, stören aber nicht.

Externe Tastatur und Maus¶

Mit einem USB On-the-go-Adapter (OTG) können externe Geräte wie Tastatur und Maus angesteckt werden. Mehrere Geräte gleichzeitig sind mit einem USB-Hub möglich. Mehr Informationen zur Nutzung einer Maus befinden sich unter Erweiterte_Konfiguration.

UKW-Radio¶

UKW-Radio kann derzeit nur im Werksmodus mit drei hartkodierten Frequenzen gehört werden. Eventuell sind diese nicht in jeder Region mit einem Sender in Reichweite belegt.
